# **Jogo da Velha**

Um jogo da velha clássico desenvolvido com **HTML**, **CSS** e **JavaScript**.

## **Sobre o projeto**
Este projeto consiste em uma implementação simples e funcional do jogo da velha (Tic-Tac-Toe), onde dois jogadores se alternam entre X e O até que haja um vencedor ou empate.

## **Funcionalidades**
• Alternância automática entre jogadores (X e O)

• Detecção de vitória

• Destaque visual para o vencedor

• Exibição do jogador atual

• Botão de reinício do jogo


## **Tecnologias utilizadas**

• HTML

• CSS

• JavaScript
